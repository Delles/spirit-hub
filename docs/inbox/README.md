# SpiritHub Inbox

Place user-provided project inputs here when they should become part of the operating record.

Use dated filenames:

```text
YYYY-MM-DD-source-topic.ext
```

Examples:

```text
2026-06-11-search-console-queries.csv
2026-06-11-vercel-analytics-top-pages.png
2026-06-11-adsense-status.png
```

Codex should review this folder during owner sessions and summarize important findings into dated
review documents under `docs/`.
